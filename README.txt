A simple dashboard widget to Whois lookup domain name registration information.

== Description ==

If you need a simple dashboard widget to lookup domain name information regularly this is a great addon! Also includes shortcode functionality for use on posts and pages with [whois]

#### How it works:

After installation / activation you will have a new widget within your WordPress dashboard, that's it!

== Installation ==

This section describes how to install the plugin and get it working.

Manual installation:

1. Download.
2. Upload `WhoisDashboardWidget` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Checkout your new dashboard widget!

Better yet:
Within your WP dashboard,

1. Mouse over "Plugins" and click "Add New".
2. Search "Whois Dashboard Widget".
3. Hit "Install Now".
4. Activate the plugin on the Plugin dashboard
5. Once complete, browse to the Dashboard tab to see your new widget!
6. Profit

== Frequently Asked Questions ==

= Does it check domain availability? =

A "Click Here" button is included to check the availability at Godaddy.

= Can I display a whois search anywhere? =

Yes you can. This plugin is based off of Powie's WHOIS.
Display the Whois search anywhere via shortcode [whois]
